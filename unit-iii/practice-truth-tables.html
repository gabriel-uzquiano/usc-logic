2. Tautologies

A formula is a tautology if and only if it is true under every assignment of truth values. A counterexample is an assignment under which the formula is false. Click Evaluate to check; counterexample rows will be highlighted in red if the formula is not a tautology.

3. Logical equivalence

Two formulas are logically equivalent if they have the same truth value on every row. Click Evaluate to check; rows where the truth values differ will be highlighted.

4. Validity
